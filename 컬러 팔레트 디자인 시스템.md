## 컬러 사용 가이드

### Primary (`#0071E3`)
- CTA(Call-to-Action) 버튼의 배경색
- 주요 링크 텍스트
- 강조 카드 또는 배너의 배경
- 활성 상태(active/selected) 표시
- **사용 비율**: 전체 화면의 약 15%

### Secondary (`#F5F5F7`)
- 페이지 전체 배경색
- 섹션 간 구분을 위한 교차 배경
- 비활성 버튼 배경
- **사용 비율**: 전체 화면의 약 70%

### Accent (`#FF2D55`)
- 프로모션/이벤트 강조 요소
- 알림 뱃지, 할인 태그
- 시선을 끌어야 하는 장식 요소
- **사용 비율**: 전체 화면의 약 2% (절제하여 사용)

### Text Primary (`#1D1D1F`)
- 페이지 제목 (h1, h2)
- 본문의 핵심 텍스트
- 네비게이션 메뉴 항목

### Text Secondary (`#6E6E73`)
- 부제목, 설명 텍스트
- 날짜, 메타 정보
- 폼 라벨

### Text Muted (`#86868B`)
- 플레이스홀더 텍스트
- 비활성 상태 텍스트
- 캡션, 각주

### Border (`#D2D2D7`)
- 아웃라인 버튼 테두리
- 카드 테두리
- 입력 필드 테두리

## 반응형 고려사항

### 다크모드 대응
```css
@media (prefers-color-scheme: dark) {
  :root {
    /* Primary Colors - 다크모드에서 약간 밝게 */
    --color-primary: #2997FF;
    --color-primary-light: #64B5F6;
    --color-primary-dark: #0071E3;

    /* Secondary Colors */
    --color-secondary: #1D1D1F;
    --color-accent: #FF375F;

    /* Background Colors - 어두운 배경 */
    --color-bg-primary: #000000;
    --color-bg-secondary: #1D1D1F;

    /* Text Colors - 밝은 텍스트 */
    --color-text-primary: #F5F5F7;
    --color-text-secondary: #A1A1A6;
    --color-text-muted: #86868B;

    /* Interactive Colors */
    --color-button-primary: #2997FF;
    --color-button-hover: #64B5F6;
    --color-link: #2997FF;
    --color-link-hover: #64B5F6;

    /* Border Colors */
    --color-border: #424245;
    --color-border-light: #2D2D2D;
  }
}
```

### 접근성 (Accessibility)
- **텍스트 대비**: `#1D1D1F` on `#FFFFFF` = 대비율 17.4:1 (WCAG AAA 통과)
- **링크 대비**: `#0071E3` on `#FFFFFF` = 대비율 4.6:1 (WCAG AA 통과)
- **보조 텍스트**: `#6E6E73` on `#FFFFFF` = 대비율 4.6:1 (WCAG AA 통과)
- 중요한 정보에는 색상 외에 아이콘이나 밑줄 등 추가 시각 요소 병행 권장

### 디자인 특징 요약
- **미니멀 & 클린**: 넓은 화이트 스페이스와 뉴트럴 배경으로 콘텐츠 집중
- **단일 포인트 컬러**: 블루 하나로 시선 유도, 과도한 색상 사용 지양
- **고급스러운 톤**: 순흰색(`#FFF`) 대신 약간의 웜 그레이(`#F5F5F7`)로 부드러운 인상
- **명확한 위계**: 텍스트 3단계(Primary/Secondary/Muted)로 정보 계층 구분
